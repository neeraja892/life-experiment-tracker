Life Experiment Tracker

A simple web app to track daily habits and analyze how they affect mood, productivity, and focus.

Features

- Add daily experiments
- Track mood, productivity, and focus
- Add notes
- Works on mobile and desktop

How to Use

1. Open the app
2. Enter experiment details
3. Save and track daily

Tech Used

- HTML
- CSS
- JavaScript
